# Winter-Fashion-Pricing-Intelligence-Report
This project analyzes winter fashion trends using a curated dataset of brands, categories, materials, prices, ratings, and trend status. The study identifies pricing patterns, brand positioning, and consumer value perceptions, offering insights into revenue potential and market dynamics.
# Problem Statement
Fashion brands need to understand how winter clothing attributes such as style, material, brand, and price—affect customer ratings, popularity, and trend status. However, these relationships are not always obvious, making it difficult to optimize pricing, product design, and marketing strategies.
This project analyzes winter fashion items to uncover patterns that influence consumer interest and determine which product features are associated with trending or high performing items.
# Tools Used
Microsoft Excel – Data cleaning, PivotTables, summary statistics, and visualization.
